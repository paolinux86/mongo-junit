## Requirements

* JDK 1.8 or newer
* Gradle
* IntelliJ IDEA

## Environment setup (IntelliJ IDEA)

* Checkout the code:
```
git clone http://github.com/paolinux86/mongo-junit
```
* Open IntelliJ IDEA and import the project as a Gradle project
* Start editing the code
